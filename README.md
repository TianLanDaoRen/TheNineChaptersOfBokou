## 📜 伯叩总纲 (The Bokou Manifesto)

> **“易简，而天下之理得矣；天下之理得，而成位乎其中矣。”**

《伯叩》之法，取“扑克”之音译，承“伯仲”之宗位。它是 **云笥散人** 在博览梅花易数、六爻纳甲、小六壬等古法的基础上，针对现代社会节奏与认知习惯进行的重构。

扑克五十有四，暗合周天历法；四色分治，演化四象乾坤。本系统摈弃了繁琐的纳甲与天干地支，直取**易理、象数、趋势**三大核心，建立起一套 **“算法精密、断语直观、反馈神验”** 的现代术数闭环。

### 核心哲学：三易闭环
*   **简易 (Simplicity)**：四进演数，三步成盘。小白亦能观位移断吉凶。
*   **变易 (Change)**：体用剥离，灵感活化。专家可于瞬息间捕捉天机。
*   **不易 (Constants)**：回归经辞，正本清源。以《周易》古训为终极定盘。

## 🏛️ 伯叩三才盘 (The Triple-Realm Matrix)

在本法门中，万事万物皆被映射于“天、人、地”三才之中：

| 三才位 | 职能定义 | 对应关系 | 核心视角 |
| :--- | :--- | :--- | :--- |
| **天卦 (Heaven)** | 观大时局、政策、背景、定数 | 天然之物 | 环境压力与时机 |
| **人卦 (Human)** | 观我心、我行、手段、变数 | 人造之物 | 自身执行与底气 |
| **地卦 (Earth)** | 观物质、实利、结果、落地 | 生灵、活物 | 最终反馈与归宿 |
